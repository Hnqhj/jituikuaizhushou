# 鸡腿快助手 安装说明

## 适用环境
- DaVinci Resolve Studio
- Windows 10/11

## 安装方式
1. 关闭 DaVinci Resolve。
2. 将 ZIP 安装包完整解压到本地文件夹，不要直接在压缩包里运行。
3. 确认解压后的目录结构如下：
   ```text
   鸡腿快助手_v1.3.4_20260601/
   ├─ install.bat
   ├─ TimelineQC/
   ├─ 安装说明.md
   └─ 免责声明.md
   ```
4. 双击解压后根目录里的 `install.bat`，它会自动申请管理员权限并开始安装。
5. 等待复制完成。
6. 重启 DaVinci Resolve Studio。
7. 在 `Workspace` > `Workflow Integrations` 中打开 `鸡腿快助手`。

注意：只复制或单独打开 `install.bat` 没有用，`install.bat` 必须和 `TimelineQC` 文件夹放在同一级目录。

## 打包内容
- 插件本体
- `WorkflowIntegration.node`
- 内置 Python 运行时
- Pillow、numpy
- FFmpeg
- 安装说明与免责声明

## 使用说明
- 先确认当前项目已打开时间线。
- 在左侧选择时间线或文件夹。
- 先执行快速检查，再按需要执行精确检查或批量检查。
- 检查结果会同步写入时间线标记。
- 如果启用了自动更新，请把 `update_config.json` 里的 `manifest_url` 指向你在 Gitee 或其他国内 Git 平台公开托管的 `update.json`。

## 常见问题
- 如果看不到插件，请确认使用的是 Studio 版。
- 如果精确检查失败，请检查 `runtime` 目录是否完整。
- 如果首次启动较慢，通常是 Resolve 载入插件和内置运行时的正常现象。
- 如果安装脚本提示找不到 `TimelineQC`，说明没有完整解压安装包，或移动了 `install.bat`。
- 如果双击后没有弹出安装窗口，先确认系统没有拦截脚本或批处理文件，再尝试右键“以管理员身份运行”。
